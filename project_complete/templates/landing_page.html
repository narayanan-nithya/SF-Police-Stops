    • Project Description

    This project is looking at San Francisco Police Department (SFPD) public data set from 2015 to examine traffic stops and arrests in relation to race and gender in Bay Area. We use coding and programing languages such as Java Script and Python to analyze as well as visualize the data set. The goal of this project is to have the most current and accurate updated information on display to inform the public about SFPD.

    Our findings indicate that while most incidents are related to white people, black people have the most arrest counts follow by white and hispanic.
